# resident-utils-rs

Resident program library.

[Documentation](https://docs.rs/resident-utils)

- execute cron loop task
- execute worker task
- ctrl+c graceful stop
- data holder for cache
- retry with timeout

## Features
### postgres
- deadpool_postgres

### redis
- deadpool_redis

### sqlx
- sqlx
